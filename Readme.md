
### Installation

```
> cd virtual-desktop-indicator/

> plasmapkg2 -i .
```


### Development

* plasmoid managment
```
> cd virtual-desktop-indicator/

> plasmapkg2 -i .  # install
> plasmapkg2 -u .  # update
> plasmapkg2 -r .  # remove
```

* restart PLASMA shell to reload changes
```
> killall plasmashell; kstart5 plasmashell
```
